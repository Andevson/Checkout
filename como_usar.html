Como usar


Sobre o software

Checkout é um software de conferência de mercadoria, que recebe como entrada um pedido em formato de tabela, e que pode ser utilizado com um leitor de código de barras.

Os pedidos são copiados de qualquer tabela que esteja em fomato de texto, como no Excel, por exemplo.


Passo a passo

  1. Configurações iniciais

    1. Através da barra de menu superior, selecione 'Opções' e em seguida 'Configurações'.

      Será aberta a tela 'Configurações', que permitirá definir parâmetros necessários para identificar corretamente os produtos de um pedido.

    2. As opção 'Usar cabeçalho' e 'Usar rodapé' são úteis para trabalhar com diferentes fontes de dados, contendo ou não cabeçalho e/ou rodapé.

      Caso a tabela importada possuia cabeçalho, é necessário marcar a opção 'Usar cabeçalho', ou o software fará a leitura do cabeçalho como se fosse um produto. O que ocasionará um erro.

      Caso a tabela importada possuia rodapé, é necessário marcar a opção 'Usar rodapé', ou o software fará a leitura do rodapé como se fosse um produto. O que ocasionará um erro.

      Atenção: não marque as opções caso não seja necessário, pois o uso indevido dessas opções pode fazer com que o alguns produtos não sejam lidos.

    3. Os campos 'Ordem ID' e 'Ordem quantidade' são úteis para trabalhar com diferentes fontes de dados, podendo reconhecer em quais colunas da tabela os atributos dos produtos estão inseridos.

      Atribua ao campo 'Ordem do ID', a posição em que o 'ID' do produto se encontra na tabela. Caso o campo informado não seja a posição adequada, o software fará a leitura da coluna como se fosse o 'ID' do produto. O que ocasionará um erro.

      Atribua ao campo 'Ordem da quantidade', a posição em que a quantidade do produto se encontra na tabela. Caso o campo informado não seja a posição adequada, o software fará a leitura da coluna como se fosse a quantidade do produto. O que ocasionará um erro.

  2. Cadastre um produto

    1. Abra a tela de 'Cadastro de produtos', clicando no botão 'Cadastro de produtos'.

    2. Insira um 'ID' para o produto, no campo 'ID'.

      Um 'ID' é um código único dado a cada produto, usado para que o produto possa ser encontrado no sistema.

    3. Insira ou escaneie o 'Código de barras' do produto, no campo 'Código'.

      Esse será o código de barras escaneado durante o processo de conferência, para confirmar a conferência do produto.

    4. Insira 'Fator de quantidade' do produto, no campo 'Fator'.

      O 'Fator de quantidade' é a quantidade de produtos que deve constar no pedido que corresponderá a uma única unidade conferida.

      Por exemplo: se o produto que está sendo conferido é uma caixa com 6 pares de chinelos e no pedido está descrito a venda individual de cada um dos chinelos, o 'Fator de saída' deve ser 6, para que você escaneie a caixa com 6 produtos ao invés de escanear um por um.

    5. Clique em 'Cadastrar'.

      Se tudo estiver certo, uma mensagem aparecerá na tela informando que o produto foi cadastrado.

    6. Repita o processo para cadastrar quantos produtos quiser.

  3. Insira um pedido

    1. Abra a tela de 'Novo pedido', clicando no botão 'Novo pedido'.

    2. Insira a tabela contendo do pedido, contendo os produtos e a quantidade de cada produto.

      Cada produto deve estar em uma linha da tabela, devendo ser informado na 'Tela de configurações' a existência ou não de cabeçalho e/ou rodapé.

      Cabeçalho e rodapé são ignorados pela conferência, já que não contêm produtos em si.

      Cada linha contendo produto deve possuir uma coluna contendo o 'ID' do produto previamente cadastrado, e uma coluna contendo a quantidade, que deve ser múltipla do 'Fator do produto'.

    3. Clique em 'Obter produtos'.

      Os produtos do pedido poderão ser visualizado em uma tebela na mesma tela.

      Após aparecerem os produtos do pedido corretamente, clique em 'Finalizar Pedido', para passar para a etapa de conferência.

    4. Se tudo estiver certo, será aberta a tela 'Pedido', onde será realizada a conferência dos produtos.

  4. Faça a conferência dos produtos

    1. Após inserir um pedido na tela 'Novo pedido', será aberta a tela de conferência de produtos.

    2. Insira ou escaneie o código de barras no campo 'Entre com código de barras'.

      Se o código de barras for válido para o pedido em questão, a tela mudará para a cor verde, caso o código de barras não for válido, a tela mudará para a cor vermelho.

      Também é possível acompanhar em uma tabela na mesma tela, os produtos que ainda restam ser conferidosm, além de ser exibido a quantidade restante de produtos no canto superior direito da tela.

      Atenção: um produto só é conferido se a tela ficar verde e o pedido diminuir o produto em uma quantidade.

    3. Ao realizar a conferência de todos os produtos do pedido, uma mensagem será exibida informando que a conferência foi realizada com sucesso.

      Caso ainda restem produtos no pedido para serem conferidos, significa que a conferência não foi realizada corretamente.
