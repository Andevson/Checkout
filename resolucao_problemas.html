Resolução de Problemas

Classificação das mensagens

Axxx = Aviso - Indica que a realização de alguma operação ou que o usuário está realizando alguma atividade de forma inadequada.
Dxxx = Defeito - Indica que há um erro interno no software que precisa ser corrigido pela equipe de desenvolvimento.
Exxx = Erro - Indica que há um erro de configuração do software, podendo ser um parâmetro incorreto ou ausência de arquivos do software.

Identificação dos problemas

  • 1xx = Interno

  • 11x = Mensagens

  • D111 = "Erro ao lançar mensagem - Mensagem não identificada."

  • 2xx = Programa

  • 21x = Base de dados

  • E211 = "Erro na base de dados - Base de dados não encontrada."

  • A212 = "Base de dados criada - Uma nova base de dados foi criada."

  • E213 = "Erro ao criar base de dados - Não foi possível ler ou criar a base de dados."

  • E214 = "Erro ao criar base de dados - Não foi possível estabelecer uma conexão com a nova base de dados."

  • E215 = "Erro ao gravar produto - Não foi possível armazenar dados no banco de dados."

  • 22x = Leitura de arquivos externos

  • E221 = "Erro ao abrir arquivo - Imagem de fundo não encontrada."

  • E222 = "Erro ao abrir arquivo - Licença não encontrada."

  • E223 = "Erro ao abrir arquivo - Manual de uso não encontrado."

  • 23x = Arquivo de configuração

  • E231 = "Erro nas configurações - Arquivo de configurações não encontrado."

  • A232 = "Configurações criadas - Um novo arquivo de configurações foi criado."

  • E233 = "Erro ao criar configurações - Não foi possível ler ou criar as configurações."

  • E234 = "Erro ao criar configurações - Não foi possível estabelecer uma conexão com as novas configurações."

  • E235 = "Erro ao carregar configurações - As configurações são inválidas."

  • 24x = Produto

  • E241 = "Erro ao carregar produto - O produto carregado da base de dados é invalido."

  • 3xx = Entrada

  • 31x = Cadastro de produto

  • A311 = "Não foi possível cadastrar o produto - Nenhum ID foi inserido."

  • A312 = "Não foi possível cadastrar o produto - ID não pode conter espaços."

  • A313 = "Não foi possível cadastrar o produto - Nenhum código foi inserido."

  • A314 = "Não foi possível cadastrar o produto - Código não pode conter espaços."

  • A315 = "Não foi possível cadastrar o produto - Nenhum fator foi inserido."

  • A316 = "Não foi possível cadastrar o produto - Fator não pode conter espaços."

  • 32x = Pedido

  • A321 = "Pedido vazio - Não há nenhum produto no pedido."

  • A322 = "Checkout concluido - Todos os produtos tiveram saída."

  • F323 = "Não foi possível acrescentar o produto - A quantidade de entrada e/ou do produto são inválidas."

  • 33x = Saída de produto

  • A331 = "Código não inserido - Nenhum código de barras foi inserido."

  • A332 = "Código inválido - Código de barras não pode conter espaços."

  • F333 = "Saída não efetuada - O pedido atual está vazio."

  • 34x = Inserção de dados

  • A341 = "ID não inserido - Nenhum ID foi inserido."

  • A342 = "ID inválido - ID não pode conter espaços."

  • A343 = "Nenhum produto inserido - A entrada não possui produtos ou os produtos foram lidos como cabeçalho ou rodapé."

  • A344 = "Quantidade não inserida - Nenhuma quantidade foi inserida."

  • A345 = "Quantidade inválida - Quantidade não pode conter espaços."

  • A346 = "Quantidade inválida - Quantidade deve ser múltipla do fator do produto."

  • A347 = "Quantidade inválida - Quantidade não pode ser nula ou negativa."

  • 35x = Configurações

  • A351 = "Configurações salvas - As configurações foram armazenadas."

  • A352 = "Configurações inválidas - A ordem do ID não pode ser a mesma da quantidade."

  • A353 = "Configurações criadas - Novas configurações foram criadas."
